# Notes
[Ngrok](https://ngrok.com/) is an awesome tool to expose an internet-accessible service, using software running on your local machine (behind a firewall.)

Help
`./ngrok help`

Start a HTTP tunnel forwarding to your local port 8000.
`./ngrok http 8000`
